# Web Programming : Goal Tracker

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Running the apllication

1. Create a new folder with any name
2. Run this code and wait for installation:
### `npx create-react-app .`
3. Install dependencies :
	- npm i bootstrap
	- npm install @mui/material @emotion/react @emotion/styled
	- npm i @mui/icons-material
	- npm install react-bootstrap
	- npm install moment --save 
	- npm i react-router-dom
	- npm install jquery popper.js

4. Download files from this repository
5. Copy and paste the following folders/files and put it in your project's 'src' folder :
  - Folders : components, data, pages, styles
  - Files : App.js

6. Run the application 
### `npm start`
